Music Movies API

REST API for an application called My Music Movies that interacts with a database storing data about movies and videos related to music. The Music Movies API will provide users with access to information about different music movies and a wealth of various related subcategories. Users will be able to sign up, update their personal information, and create a list of their favorite music movies.

Essential Features:

Business Logic URL[Query Parameters] HTTP Method Request body data format Response body data format
Return a list of ALL movies /movies GET None A JSON object holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing an ID, artist, title, yt_url, description, genre, director, image URL, featured. Example:
{
id: 1,
artist: 'Pink Floyd',
title: 'Pulse',
description: 'Lorem description',
genre: 'ProgRock',
director: 'David Mallet',
image_url: null,
yt_url: 'https://www.youtube.com/watch?v=HriYRoxWo1I',
featured: true
}
Return data about a genre (description) by name/title (e.g., “MelodicTechno”) /movies/genres/[Genre Name] GET None A JSON object holding data about all movies tagged with the genre "Melodic Techno"
Return data about a director (bio, birth year, death year) by name /movies/directors/[Director Name] GET None A JSON object holding data about a director. Example:
{
"name": "Adrian Maben",
"bio": 'Adrian Maben is a director and writer, known for Pink Floyd at Pompeii (1972)',
"birthyear": 1942,
"deathyear": null
}
Allow new users to register /users POST A JSON object holding data about the user to add, structured like:
{
"name": "Hans-Peter Mustermann",
"favoriteMovies": []
}
 A JSON object holding data about the user that was added, including an id:
{
"id": "f133978c-21f9-4196-b881-2213517adb43",
"name": "Hans-Peter Mustermann",
"favoriteMovies": []
}
Allow users to update their user info (username) /users/[ID] PATCH A JSON object holding data about the updated user info:
{
"name": "HP Mustermann"
}
 A JSON object holding a complete data set about the user that was updated, including the updated username:
{
"id": "f133978c-21f9-4196-b881-2213517adb43",
"name": "HP Mustermann",
"favoriteMovies": []
}
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/[ID]/[NewMovieTitle] POST None A text message indicating whether the movie has been successfully added
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed) /users/[ID]/[MovieTitle] DELETE None A text message indicating whether the movie has been successfully removed
Allow existing users to deregister (showing only a text that a user email has been removed) /users/[ID]/ DELETE None A text message indicating whether the user has been successfully deleted
