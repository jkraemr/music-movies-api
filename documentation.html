Music Movies API

The Music Movies API will provide users with access to information about different music movies and a wealth of various related subcategories. Users will be able to sign up, update their personal information, and create a list of their favorite music movies.

Essential Features:

  • REST API for an application called My Music Movies that interacts with a database storing data about moving image content related to music
  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Documentary”)
  • Return data about a artist etc. (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info (username, password, email, date of birth)
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister
